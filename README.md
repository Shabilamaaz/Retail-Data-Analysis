# Retail Sales Analysis (Python)

This project analyzes the Superstore dataset using Python. It includes data cleaning, feature engineering, and visualizations to understand sales patterns, profit trends, delivery time, and customer behavior.

---

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- VS Code
- Superstore Dataset (Kaggle)

---

## Key Steps
- Loaded and cleaned the dataset  
- Removed duplicates and handled date formats  
- Created new features like profit margin, delivery days, revenue per unit  
- Generated charts for sales, profit, top sub-categories, and region analysis  

---

## Output Files
- `analysis.py`
- `plots/` (folder with charts)
- `retail_with_features.csv`

---

1. Install required libraries:
   ```bash
   pip install pandas matplotlib seaborn
